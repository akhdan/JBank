/** Ketika mengetik suatu comment, untuk membuat baris baru dapat dilakukan dengan menekan enter

/** biasanya single line comment digunakan, bila ingin mendeskripsikan detail suatu method berupa tag */
/** Tag tersebut terdiri sebagai berikut :
 * @author : nama dari penulis
 * @version : versi program

 * @return : nilai yang dikembalikan pada method
 * @see : untuk memperlihatkan referensi
 * dsb
